# Сумма отрицательных элементов между минимумом и максимумом массива
#Описание задачи
Дан одномерный массив `A` размерности `N`.  
Нужно найти сумму отрицательных элементов, расположенных между минимальным и максимальным элементами массива (исключая сами min и max).
Пример входных данных
python
A = [3, -4, 5, -2, 7, 8, -1, 6]
- Максимум: 8 (index 5)
- Минимум: -4 (index 1)
- Элементы между ними:[5, -2, 7]
- Отрицательные среди них: -2
- Результат: -2
#Логика алгоритма
1. Находим индекс максимального и минимального элемента массива.
2. Определяем элементы между ними.
3. Отбираем только отрицательные значения.
4. Возвращаем их сумму.
